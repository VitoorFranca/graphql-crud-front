> # Installation

## FrontEnd

```sh
git clone git@github.com:VitoorFranca/graphql-crud-front.git

cd graphql-crud-front

yarn install
```

## BackEnd ([Repository](https://github.com/VitoorFranca/graphql-api)).

```sh
git clone git@github.com:VitoorFranca/graphql-api.git

cd graphql-api.git

npm install
```
